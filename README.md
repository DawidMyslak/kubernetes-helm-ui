## Build it yourself

You will need Node.js in version 7 or higher installed on your machine.

``` bash
# clone repository
git clone https://github.com/DawidMyslak/kubernetes-helm-ui.git
cd kubernetes-helm-ui

# install dependencies
npm install

# build application
npm run build
```
